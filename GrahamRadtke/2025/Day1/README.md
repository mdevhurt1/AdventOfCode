# Advent of Code 2025 - Day 1 (Graham Radtke)

## Overview
This folder contains solutions for Day 1 of Advent of Code 2025 by Graham Radtke.

- `problem1.cpp`: Solves Part 1, finding how many times the dial points at 0 after each rotation.
- `problem2.cpp`: Solves Part 2, counting every click that causes the dial to point at 0 during all rotations.
- `puzzle_input.txt`: The input file containing the sequence of dial rotations.
- `problem1.txt` and `problem2.txt`: Problem descriptions for each part.

## Usage
Compile each solution with g++ and run in a bash terminal:

```bash
g++ problem1.cpp -o problem1
./problem1 < puzzle_input.txt # Prints the answer for Part 1

g++ problem2.cpp -o problem2
./problem2 < puzzle_input.txt # Prints the answer for Part 2
```

## Answers
- Part 1: 1071
- Part 2: 6700
