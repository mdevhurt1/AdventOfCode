# Advent of Code 2025 - Day 4 (Graham Radtke)

## Overview
This folder contains solutions for Day 4 of Advent of Code 2025 by Graham Radtke.

- `problem1.cpp`: Solves Part 1, finding the number of rolls of paper touching fewer than 4 other rolls of paper.
- `problem2.cpp`: Solves Part 2, finding the total number of rolls of paper touching fewer than 4 other rolls of paper with removal.
- `puzzle_input.txt`: The input file containing the map of the rolls of paper.
- `problem1.txt` and `problem2.txt`: Problem descriptions for each part.

## Usage
Compile each solution with g++ and run in a bash terminal:

```bash
g++ problem1.cpp -o problem1
./problem1 < puzzle_input.txt # Prints the answer for Part 1

g++ problem2.cpp -o problem2
./problem2 < puzzle_input.txt # Prints the answer for Part 2
```

## Answers
- Part 1: 1587
- Part 2: 8946
