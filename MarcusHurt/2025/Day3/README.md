# Advent of Code 2025 - Day 3 (Marcus Hurt)

## Overview
This folder contains solutions for Day 3 of Advent of Code 2025 by Marcus Hurt.

- `problem1.py`: Solves Part 1, finding the largest possible two-digit joltage from each battery bank and summing the results.
- `problem2.py`: Solves Part 2, finding the largest possible twelve-digit joltage from each battery bank and summing the results.
- `puzzle_input.txt`: The input file containing the battery banks.
- `problem1.txt` and `problem2.txt`: Problem descriptions for each part.

## Usage
Run each solution with Python 3:

```bash
python3 problem1.py  # Prints the answer for Part 1
python3 problem2.py  # Prints the answer for Part 2
```
