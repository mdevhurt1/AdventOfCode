# Advent of Code 2025 - Day 2 (Marcus Hurt)

## Overview
This folder contains solutions for Day 2 of Advent of Code 2025 by Marcus Hurt.

- `problem1.py`: Solves Part 1, finding and summing all invalid product IDs in the given ranges where the ID is made of a sequence of digits repeated twice (e.g., 1212, 123123).
- `problem2.py`: Solves Part 2, finding and summing all invalid product IDs in the given ranges where the ID is made of a sequence of digits repeated at least twice (e.g., 121212, 123123123).
- `puzzle_input.txt`: The input file containing the product ID ranges to check.
- `problem1.txt` and `problem2.txt`: Problem descriptions for each part.

## Usage
Run each solution with Python 3:

```bash
python3 problem1.py  # Prints the answer for Part 1
python3 problem2.py  # Prints the answer for Part 2
```
