# Advent of Code 2025 - Day 1 (Marcus Hurt)

## Overview
This folder contains solutions for Day 1 of Advent of Code 2025 by Marcus Hurt.

- `problem1.py`: Solves Part 1, finding how many times the dial points at 0 after each rotation.
- `problem2.py`: Solves Part 2, counting every click that causes the dial to point at 0 during all rotations.
- `puzzle_input.txt`: The input file containing the sequence of dial rotations.
- `problem1.txt` and `problem2.txt`: Problem descriptions for each part.

## Usage
Run each solution with Python 3:

```bash
python3 problem1.py  # Prints the answer for Part 1
python3 problem2.py  # Prints the answer for Part 2
```

## Answers
- Part 1: 1074
- Part 2: 6254
